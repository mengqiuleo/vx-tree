# 自定义配色

在引入默认配置后，引入自己的配置来进行覆盖。

```js
:root {
  --vxe-brand: red; //checkBox style
  --vxe-list-item-selected-bg: blue; //selected style
  --vxe-list-item-hover-bg: yellow; //hover style
}
```

默认颜色为:
```js
--vxe-brand: #5e7ce0;
--vxe-list-item-selected-bg: #f2f5fc;
--vxe-list-item-hover-bg: #f2f2f3; 
```
